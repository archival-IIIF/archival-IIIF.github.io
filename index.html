Archival IIIF is an open source framework to present born digital archives
Viewer Server Demo

Open source

Archival IIIF is Open Source and contributed under the MIT License.

Standards compliant

Archival IIIF implements the International Image Interoperability Framework (IIIF) and can be used as a component of an Open Archival Information System (OAIS).

Flexible

Thanks to its service system it is possible to build bridges to external ingest and authentification software. In the core version services for Archivematica and Docuteam Feeder are included.

Lightweight

Archival IIIF is designed to be as light as you want - you can pick and choose the features you want to include.
